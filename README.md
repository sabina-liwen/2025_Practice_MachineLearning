# Machine Learning Fundamentals

This repository contains introductory machine learning exercises completed in 2025 as part of my self-study (based on Andrew Ng’s Machine Learning course).

The focus is on understanding core concepts through simple implementations using Python and NumPy.

## Contents

### Linear Regression
A basic regression task to model the relationship between input features and a continuous target.

- implemented cost function
- applied gradient descent
- trained a linear model for prediction

### Logistic Regression
A binary classification task predicting admission outcomes from two exam scores.

- implemented logistic regression with NumPy
- modeled probability using the sigmoid function
- performed basic classification

## Tools

- Python
- NumPy

## Learning Focus

- understanding core supervised learning concepts  
- connecting mathematical formulation with code  
- implementing simple models from scratch  

## Notes

This repository represents early-stage practice and focuses on conceptual clarity rather than complexity or optimization.
